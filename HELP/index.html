File Structure

When you will unzip the archive you will find 4 folders:
  • WWW - Theme folder, this you will upload to your server
  • SASS - SASS files, and Grunt Configuration
  • PSD - Theme PSD File, if you want to make custom modifications to design
  • HELP - File you are looking right now
All you need is in WWW, this is the folder you want to upload to your server, and is having the following structure:
  • /fonts
    • FontAwesome Library
  • /images
    • Organized folders for images used in Theme
  • /inc
    • contact.php - PHP File for forms
  • /scripts
    • main.js - Main jQuery File where the magic takes place
    • main.min.js - Minified version of main.js
    • plugins.js - jQuery Plugins used in Theme
    • plugins.min.js - Minified version of Plugins

Installation

To install this theme, you need a server, copy WWW folder, paste it on your server and done.

Note: After you moved WWW content on your server, you can choose one of the index-* file and rename the file in index.html

Page Section

You can add different classes with page-section class

Options:

parallax - This is a special class used on parallax sections, with this class you must use attr data-background="yourPathToImage/parallaxImage.jpg"

fullscreen - Make the section full height size ( indicated to use only on <header> )

page-section-small - Does same as page-section but with smaller margins

overlay - Add background overlay over an image 
    
    // Simple section
    <section class="page-section">
      <div class="container">
     
        // Content
    
      </div>
    </section>

    // Section with background
    <section class="page-section parallax align-center" data-background="images/parallax/parallax-bg1.jpg">
      <div class="container">
     
        // Content
    
      </div>
    </section>

Note: In config.php we have all the information we need to setup our forms

    $Use_Mailchimp = true;

    // Mailchimp API KEY
    $Mailchimp_API_Key = '123456789209318070asiuzi8134809-us1';

    // Mailchimp List ID
    $Mailchimp_List_ID = '09318070asa';

    // Mailchimp Double Opt In

    $Mailchimp_Welcome = false;

    /* =====================================================
    Contact Form
    ===================================================== */

    // Email address

    /* =====================================================
    Pricing Form
    ===================================================== */

    // Email address for Pricing Form

    /* =====================================================
    Form Event Submission
    ===================================================== */

    // Email address for Event Submission

How to find your Mailchimp API Key

  1. Login in to your Mailchimp here
  2. Click on your name / username and go to Account
  3. Click on Extras / API Keys
  4. Copy API Key and paste it on $Mailchimp_API_Key

How to find your Mailchimp List ID

  1. Click on Lists, then click on List Dropdonw Status and Settings
  2. On the end of the page you will find your List ID

Paypal Settings

Configure Paypal Form:

Options:

yourPaypalEmailAddress - Must be replaced with your email address, you will receive donations on this email

name="item_name" - The value of this field is the title of your Paypal after submit form

name="currency_code" - You can change the value in your currency ( EUR, GBP, etc )

        <input type="hidden" name="no_note" value="1">

Google Map

In this section we will learn how to set up google map and some options

Note: You can only have 1 google map per page

Options:

  • style="300px" - You can set a custom height for your map
  • data-address=" " - Your location ( Prince Albert, SK, Canada )
  • data-address-title=" " - Title for Map infoWindow
  • data-address-id=" " - Where ( address-info ) is the trigger for your custom infoWindow
The other options like data-restaurant, data-restaurant-title, data-restaurant-id works in the same system like address 
    
    // Map Section
    <section id="location" class="google-map page-section">
        <div class="container">
            <div class="row">
                <div class="c-5 offset-7">

                // Custom Info Window
                <div class="address-info custom-infowindow active">
                  <h3>Event Location</h3>
                  <p>87540 North Main Street, <br> New York, NY 98139</p>
                  <p>Phone: 1-800-555-9172</p>
                  <p>Email: <a href="mailto:support@uberevent.com">support@uberevent.com</a></p>
                </div>

                </div>
            </row>
        </container>

        // Google Map
        <div class="map"
        style="350px" // Optional
        data-address="Prince Albert, SK, Canada"
        data-address-title="Event Location"
        data-address-id="address-info"></div>
        <script src="http://maps.googleapis.com/maps/api/js?sensor=false" type="text/javascript"></script>

    </section>
If you just want Google Map without infowindows you can use: 
    
    <div class="map" style="350px" data-address="Prince Albert, SK, Canada"></div>
    <script src="http://maps.googleapis.com/maps/api/js?sensor=false" type="text/javascript"></script>

FAQ

  1. Is this an Wordpress Theme ?
    No, is just HTML version, and you can install it by copying WWW folder on your server
  2. How to remove appear animations ?
    Find and delete the class animation from <body> tag
  3. The images are included in download file ?
    No, it's forbidden to include demo images on main download file.
  4. Why contact forms or videos doesn't work ?
    This things works only on a web server
  5. What do I need to worki with Grunt ?
    You need to install grunt in SASS folder and add the following packages: grunt-contrib-cssmin, grunt-contrib-imagemin, grunt-contrib-jshing, grunt-contrib-sass, grunt-contrib-uglify, grunt-contrib-watch
